## Data
#### csv files from kaggle:

sp500_edges.csv - the edges between companies (if there is a strong correlation between 2 companies - there will be an edge between them).

## output files
degree_distribution.png - the distribution of the the degrees (P(k)).

sp500_project.gexf - the input file for gephi.
